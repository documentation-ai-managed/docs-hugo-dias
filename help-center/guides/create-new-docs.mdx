## Create a Page

<Steps>
  <Step title="Open your project dashboard">
    Navigate to your project from the Hugo Dias main dashboard.
  </Step>
  <Step title="Click 'Add Page' button">
    Find and click the **Add Page** button on the sidebar.
  </Step>
  <Step title="Enter page details">
    Provide a title and optional description for the page.
  </Step>
  <Step title="Save and edit your page">
    Click **Save** and use the editor to add content.
  </Step>
</Steps>
