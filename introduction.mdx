## Overview

Hugo Dias provides a centralized space where you organize, manage, and collaborate on project documentation. You create structured docs for APIs, user guides, and internal wikis with ease. This platform scales with your projects, offering search, versioning, and sharing features to keep teams aligned.

Use Hugo Dias to streamline documentation workflows, reduce silos, and ensure consistency across your repositories and teams.

## Quick Start

Get up and running with Hugo Dias in minutes. Follow these steps to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://hugo-dias.example.com` and create a free account. Verify your email to unlock full features.
  </Step>
  <Step title="Create a Space" icon="plus">
    Click "New Space" in the dashboard. Name it (e.g., "Project Alpha Docs") and select a template.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Navigate to your space and create a page. Use Markdown or the visual editor to write content.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Hit "Publish" to make it live. Generate a shareable link or embed code for your team.
  </Step>
</Steps>

<CodeGroup tabs="Markdown,Visual Editor">
  ```markdown
  # Project Alpha Overview

  Welcome to the documentation for Project Alpha.

  ## Quick Links
  - [Installation](/installation)
  - [API Reference](/api)
  ```
  ```html
  <h1>Project Alpha Overview</h1>
  <p>Welcome to the documentation for Project Alpha.</p>
  <h2>Quick Links</h2>
  <ul>
    <li><a href="/installation">Installation</a></li>
    <li><a href="/api">API Reference</a></li>
  </ul>
  ```
</CodeGroup>

<Callout kind="tip">
  Pro tip: Integrate with GitHub for automatic syncs. Connect your repo in Settings > Integrations.
</Callout>

## Brand Guidelines

Maintain consistency with Hugo Dias's brand. The primary color is `#3B82F6` (blue). Use it for headers, buttons, and accents.

<Tabs>
  <Tab title="Color Palette" icon="palette">
    | Color | Hex | Usage |
    |-------|-----|-------|
    | Primary | `#3B82F6` | Buttons, links |
    | Secondary | `#1E40AF` | Hover states |
    | Neutral | `#F8FAFC` | Backgrounds |
  </Tab>
  <Tab title="CSS Usage" icon="code">
    ```css
    :root {
      --primary-color: #3B82F6;
    }

    .button {
      background-color: var(--primary-color);
      color: white;
      border-radius: 6px;
      padding: 12px 24px;
    }
    ```
  </Tab>
</Tabs>

<Expandable title="Advanced Branding" default-open="false">
  Customize further with custom CSS in space settings. Upload logos up to 2MB. Preview changes live before applying.
</Expandable>

<Callout kind="info">
  Always test brand customizations across devices. Use `#3B82F6` at 100% opacity for primary elements.
</Callout>
